#### Fibonacciho trojúhelník

  * **Cíl:** Kombinace vnořených cyklů a efektivní generování posloupnosti.
  * **Úkol:** Napište program, který se uživatele zeptá na počet řádků (N). Program poté vygeneruje a vypíše N řádků Fibonacciho posloupnosti v trojúhelníkovém tvaru.
      * Řádek 1: 1 číslo (0)
      * Řádek 2: 2 čísla (1, 1)
      * Řádek 3: 3 čísla (1, 2, 1)
      * Řádek 4: 4 čísla (1, 3, 3, 1)
      * ... atd.
  * **Příklad výstupu pro N = 4:**
    ```
       1
      1 1
     1 2 1
    1 3 3 1
    ```
  * **Klíčové koncepty k procvičení:**
      * Vnořené cykly (`for`).
      * Správa stavu.
